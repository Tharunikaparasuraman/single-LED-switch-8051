# Single LED Control Using Switch (8051)

## Description
This project demonstrates how to control a single LED using a push button switch with the 8051 microcontroller. The program is written in Embedded C using Keil uVision and simulated in Proteus.

## Tools Used
- Keil uVision
- Proteus

## Components
- AT89C51 Microcontroller
- LED
- Push Button Switch
- 220Ω Resistor
- 10kΩ Resistor
- Power Supply

## Files Included
- single led with switch.c
- single led with switch.hex
- single led with switch.uvproj
- single led with switch.pdsprj
- Proteus Connection Screenshot

## Working
- Press the push button to turn the LED ON.
- Release the push button to turn the LED OFF.
- The switch controls the LED through the 8051 microcontroller.

## Output
The LED turns ON and OFF based on the push button input.
